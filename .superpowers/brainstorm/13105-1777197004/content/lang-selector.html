Onde colocar o seletor de idiomas?

Escolha a posição e estilo que prefere para o botão de idioma na navbar

A

Globo + Dropdown

Ícone de globo com o idioma actual. Clica e abre menu com as 3 opções.

Exemplo na navbar:
Atlantic Crown — Coffee —
  • Produtos
  • História
  • Contacto
B

Botões de Idioma visíveis

As 3 siglas sempre visíveis na navbar — EN / PT / FR. O idioma activo fica dourado.

Exemplo na navbar:
Atlantic Crown — Coffee —
  • Produtos
  • História
  • Contacto
C

Texto simples no menu mobile

No desktop fica discreto junto aos links. No mobile aparece no menu hambúrguer.

Exemplo na navbar:
Atlantic Crown — Coffee —
  • Produtos
  • História
  • Contacto
  • │
  • PT EN FR
